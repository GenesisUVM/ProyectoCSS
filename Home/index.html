¿Quieres aprender a hacer una torta volteada de piña como todo un profesional?

Aqui te enseñamos los tips mas importantes para lograrlo

También te regalaremos un PDF con la receta, sigue esta paso a paso y beras que es lo que siempre buscaste en una receta

Dercargar PDF
  • Ten ingredientes de buena calidad para asegurar
  • Asegurate de tener todos los materiales que necesitaras
  • Pesa todo antes de empezar a mezclar los ingredientes
  • Al picar las piñas asegurate de que las rodajas no sean muy gruesas
  • Espera que el caramelo este totalmente frio para colocar las piñas
  • Recuerda el polvo de hornear pero no abuses de el o tendrá el efecto contrario
  • No llenes demasiado el molde, recuerda que el volumen de las piñas harán que necesites menos mezcla
  • Sacale el aire antes de meter tu torta al horno
  • Asegurate que el horno este bien caliente y a un temperatura adecuada para que tenga una cocción pareja
  • Deja enfriar totalmente la torta
  • Corta la panza de la torta si creció dispareja para que sea más fácil desmoldar
  • Calienta un poco en la hornilla para que el caramelo se despegue de forma perfecta
